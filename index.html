🌐 Host Online (GitHub Pages)

✅ Free hosting — one link for your whole team. Takes 5 minutes.
1

Create a GitHub account

Go to github.com → Sign Up (free). Use your company email.

2

Create a new repository

Click + → New repository → Name it peb-system → Set to Public → Create repository

3

Upload this HTML file

In your repository → Click Add file → Upload files → Upload PEB_Final_WebApp.html → Rename it to exactly:

index.html
4

Enable GitHub Pages

Repository → Settings → Pages → Source: main branch → Save

Your live link will be:

https://YOUR-USERNAME.github.io/peb-system
5

Share the link

Send this one link to every team member. They open it in any browser — phone, laptop, anywhere. No installation needed.

⚠️ Important after hosting
Each team member opens the link → pastes the Google Apps Script URL once → selects their name → they're connected. Settings are saved in their browser.
